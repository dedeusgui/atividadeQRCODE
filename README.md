# SENAI QR Code

Aplicacao web criada com `React + Vite + TypeScript` para gerar QR Codes a partir de links. O projeto foi desenhado com uma interface minimalista, responsiva e inspirada na identidade visual do SENAI.

## Funcionalidades

- Geracao de QR Code a partir de um link
- Validacao de URL antes da geracao
- Preview do QR Code na tela
- Download do QR Code em formato PNG
- Layout responsivo para desktop e mobile

## Tecnologias

- React
- Vite
- TypeScript
- `qrcode`
- `@fontsource/sora`
- `@fontsource/manrope`

## Como rodar localmente

```bash
npm install
npm run dev
```

## Build de producao

```bash
npm run build
```
